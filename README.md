# Lab 3: VirtualBox Guest Additions and Local Configuration

## Overview
This lab documents installing VirtualBox Guest Additions, creating a local user account, and configuring a shared folder on a Windows virtual machine in a lab environment.

## Tools Used
- Oracle VirtualBox
- Windows virtual machine
- Host OS: Windows 11

## Lab Objectives
- Install VirtualBox Guest Additions
- Create a local user account
- Configure a shared folder between host and virtual machine

## Steps Performed
1. Inserted and installed VirtualBox Guest Additions on the virtual machine
2. Rebooted the system to complete installation
3. Created a local user account
4. Configured a shared folder between the host and virtual machine
5. Verified Guest Additions functionality and shared folder access

## Result
- Guest Additions installed successfully
- Local user account created
- Shared folder accessible from the virtual machine

## Notes / What I Learned
- Guest Additions improves usability in virtual machines
- Shared folders allow file transfer between host and VM
- Local account creation is a common basic configuration task
